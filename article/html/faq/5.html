在做网页开发的时候可能会遇到需要段落空两个字符,来整体美化页面排版.可以通过设置p标签的首行缩进来实现.

1 CSS设置首行缩进

可以使用CSS中的text-indent属性设置来实现首行缩进2字符的效果.以下示例为p标签首行缩进2个字符:

1.1 CSS设置首行缩进2个字符效果

css示例代码:

/*大屎蛋教程网 http://dashidan.com*/
p {
    text-indent:2em;
}

1.2 CSS设置首行缩进像素值

css示例代码:

p {
    text-indent: 10px;
}

1.2 CSS设置首行缩进百分比

css示例代码:

p {
    text-indent: 10%;
}
